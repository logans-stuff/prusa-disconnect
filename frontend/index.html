About PrusaDisconnect

Self-hosted Prusa Connect alternative. Talks directly to your printers via PrusaLink's local REST API. No cloud, no internet required.
Version 0.2.0    API PrusaLink v1    Polling 3s    Camera 15s

PrusaSlicer Integration

PrusaDisconnect is compatible with PrusaSlicer's network upload. Configure it as an OctoPrint printer:
Host —
API Key prusadisconnect (any non-empty value works)
In PrusaSlicer: Printer Settings > Physical Printer > Host Type: OctoPrint

Camera Setup

Cameras work two ways:
1. PrusaLink camera — Automatically detected if your printer has a camera connected via PrusaLink.
2. External URL — Set a camera URL (MJPEG snapshot endpoint) when adding/editing a printer. Works with mjpg-streamer, motionEye, etc.
